# Copywriting Skill

## Hook formulas (choose one)

- Curiosity + specificity: “Most people miss X. Here’s the number.”
- Contrarian: “X is overrated. Y is the real lever.”
- Pattern interrupt: “Stop doing X. Do this instead.”

## CTAs (pick one)

- Question that forces a trade-off
- “What would you do?” with a concrete scenario
- “If you’re doing X, try Y this week.”
