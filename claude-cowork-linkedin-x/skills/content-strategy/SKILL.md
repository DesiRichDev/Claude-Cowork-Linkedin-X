# Content Strategy Skill

## Pillar mapping

For any signal, map it to a single pillar and choose a framework:

- STORY: event → tension → lesson
- LESSON: concept → breakdown → application
- EDU-TELLING: example → principle → takeaway
- YOU: direct challenge → reframe → action

## Angle selection

Pick one angle only:

- “What this changes”
- “Why the obvious take is wrong”
- “The step everyone skips”
