# Reverse Engineering Skill

## What to extract

- Hook structure (first 2 lines)
- The “why now” angle
- The core claim + proof
- The pacing (beats / turns)
- The CTA style

## Output

Return patterns as reusable templates:

- Hook template: `[...]`
- Beat map: `1) ... 2) ... 3) ...`
- Proof types used: `data / authority / contrast / example`
- CTA template: `[...]`
